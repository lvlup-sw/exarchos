## Usage Guidelines

### Always use (Tier 1 terms)
- Durable workflows
- Convergence gates
- SDLC structure
- Audit trail
- Agent teams

### Avoid as lead terms
| Avoid | Use Instead |
|:------|:------------|
| CMDP / Constrained MDP | "structured workflows" or "convergence gates" |
| HSM / Hierarchical State Machine | "phase-gated workflow" |
| D1-D5 (numbered dimensions) | Name the dimensions explicitly |
| Governance (as lead term) | "structure" or "quality verification" |
| Enforcement (as lead term) | "verification" or "quality gates" |

### Tone
- Direct and technical — no hype, no superlatives
- Lead with concrete capabilities, not abstract promises
- Acknowledge trade-offs honestly (e.g., learning curve exists)
- Target: developer who has used Claude Code for 2+ weeks and felt the pain
